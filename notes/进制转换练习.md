**9. 进制转换应用题**

(a) 假设一个计算机内存地址是 `1A3F`（十六进制），请将其转换为二进制和十进制。

```
0x1A3F
0b 0001  1011  0011 1111
15 + 48 + 2560 + 4096 = 6719
```

```
0o234

4 + 24 + 128 = 156
```
